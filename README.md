## ⚙️ Como rodar

1. Clone o repositório:
   ```bash
   git clone https://github.com/Eng-Soft-Claudio/ExerciciosPython.git
   cd ExerciciosPython
   python nome_do_arquivo.py
   ```
Certifique-se de ter o Python 3 instalado!
